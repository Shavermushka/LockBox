# Менеджер паролей

Кроссплатформенное приложение для безопасного хранения и управления паролями с графическим интерфейсом на PyQt6.

## Возможности

- 🔐 Безопасное шифрование паролей (Fernet, мастер-пароль)
- 📂 Хранение в SQLite
- 🔍 Поиск и фильтрация по категориям
- 🎲 Генератор надёжных паролей
- 📊 Статистика по категориям (график)
- 🌐 Сканер портов (инструмент)
- 📤 Экспорт/импорт в JSON, CSV, Excel, YAML
- 🎨 Поддержка светлой и тёмной темы
- ⚡ Многопоточность для длительных операций

## Требования

- Python 3.10+
- PyQt6
- Другие зависимости указаны в `requirements.txt`

## Установка и запуск

1. Клонируйте репозиторий или скачайте архив.
2. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux
   # или venv\Scripts\activate  # Windows
```

1. Установите зависимости:

```
pip install -r requirements.txt
```
1. Запустите приложение:

```
python main.py
```

При первом запуске будет создана база данных в `~/.password_manager/passwords.db`.

## Использование

- **Добавление записи**: нажмите "Добавить" или кнопку на панели инструментов.
- **Редактирование**: дважды кликните по строке или используйте контекстное меню.
- **Просмотр пароля**: нажмите "Показать" в колонке "Действия".
- **Поиск**: введите текст в поле поиска.
- **Экспорт/импорт**: используйте соответствующие кнопки.
- **Генерация пароля**: перейдите на вкладку "Генератор".
- **Сканер портов**: перейдите на вкладку "Инструменты".
- **Смена темы**: в меню "Настройки" или через кнопку на панели.

## Настройка мастер-пароля

В текущей версии мастер-пароль задаётся в коде (для упрощения). В реальном приложении следует добавить диалог входа. Для тестирования можно установить пароль в `main.py` или через настройки.

## Сборка исполняемого файла (опционально)

Установите PyInstaller:

```
pip install pyinstaller
```

Затем выполните:

```
pyinstaller --onefile --windowed main.py --name password_manager
```

Исполняемый файл появится в папке `dist`.

## Структура проекта

- `main.py` – точка входа
- `src/` – исходный код

- `main_window.py` – главное окно
- `ui/` – диалоги и виджеты
- `logic/` – бизнес-логика (БД, шифрование, менеджер)
- `utils/` – вспомогательные функции
- `resources/` – стили и иконки
